Quick Start

  1. Assemble the Device
    Use the 3D printed parts, a buzzer, a button, and some wires. Secure the parts using iron thread—holes are provided near the buzzer and button for this purpose.

  2. Build the Headband & Place Electrodes
    You'll need 3 electrodes:
    • 1 ground electrode — place it in the center of your forehead, away from the temples.
    • 2 signal electrodes — place them symmetrically near your temples.

    To find the exact muscle location, lightly clench your jaw and feel where the temporalis muscle contracts under your fingers.

    Make sure to:
    • Move hair out of the way.
    • Use conductive gel or replacement pads—dry pads won't work reliably.

    Fix everything in place with an elastic band (like the ones for underwear). Make some holes and have the electrodes clip through it.
    You might want to print a clip in TPU like this one (resize the model appropriately).

  3. Install the Arduino IDE
    Install the CMSIS-DSP library from the Library Manager.

  4. Get the Arduino Code
    Download the latest Arduino sketch from the GitHub repository.

  5. Configure Wi-Fi
    Edit the file user/WifiSettings.h with your network credentials, then upload the sketch to the Arduino UNO R4 WiFi using the Arduino IDE.

  6. Install the Android App
    Download the latest release APK from GitHub Releases and install it on your device.
    Launch and grant all requested permissions.

  7. Start the System
    Wear the headband, power the device, and long-press the button for 4 seconds—you’ll hear a beep sequence to confirm it's streaming the tuning data.
    Note: this only works during the first 2 minutes of runtime. If it doesn't work, restart the Arduino device.

  8. Tune the Threshold
    The app will automatically detect the signal and enter a tuning screen. Adjust the classification threshold so that it's triggered only during real clenching events. When satisfied, tap “Awesome”.

    Long-press the button for 4 seconds to disable tuning. If you don't hear any confirmation, restart the Arduino device.

  9. Customize Tags
    Go to the “Tags” section of the app to select or add pertinent tracking elements.

  10. Done!
    You’re all set to begin tracking your bruxism events!
