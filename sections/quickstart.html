Welcome

This GitHub page hosts an easy-to-consult guide to build, understand, and use the open-source Bruxism Detector. It's themed with Material3 colors for clarity and accessibility. Navigate through the sections using the menu above.

Safety Notice

NEVER wear electrodes while the circuit is connected to any wall-powered source (USB to PC, charger, or power bank charging from wall). Always use a battery-only powered setup when using electrodes.
