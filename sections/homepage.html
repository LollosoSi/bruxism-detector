Welcome

This GitHub page hosts an easy-to-consult guide to build, understand, and use the open-source Bruxism Detector.
The goal is to help you detect bruxism events, interrupt them with a gentle alarm, and track data to discover the root causes.

Use the menu to explore detailed instructions on hardware setup, machine learning, Android integration, and customization options.
Each section is crafted for ease of use..

Safety Notice

NEVER wear electrodes while the circuit is connected to any wall-powered source (USB to PC, charger, or power bank charging from wall). Always use a battery-only powered setup when using electrodes.
